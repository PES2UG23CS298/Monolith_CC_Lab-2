## Overview
This project demonstrates a **Monolithic Application Architecture** using **FastAPI** through a college fest management system.  
All features such as registration, login, events, checkout, and my-events are implemented in a **single codebase and deployed as one unit**.

The lab highlights:
- Advantages of monolithic architecture
- Single point of failure
- Bug fixing in a monolith
- Performance testing and optimization using Locust

---

## Tech Stack
- **Backend Framework:** FastAPI
- **Language:** Python 3
- **Database:** SQLite
- **Load Testing Tool:** Locust
- **Server:** Uvicorn
